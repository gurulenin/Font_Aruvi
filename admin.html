📋 Workflow Guide

  1. Drop your font files into the fonts/ directory
  2. Run ./generate-manifest.sh > fonts.json to create the initial manifest
  3. Come here → upload a CSV with metadata to fill in names, licenses, designers
  4. Optionally edit entries inline in the table above
  5. Click Export fonts.json → download the updated file
  6. Replace fonts.json in your project → push to git → Netlify auto-deploys
CSV Format:
filename,display_name,family,category,license,designer,foundry,year,description,tags
Eg: NotoSerifTamil.ttf,Noto Serif Tamil,Noto Serif Tamil,Serif,OFL,Google,Google,2020,Classic serif font,serif;google
